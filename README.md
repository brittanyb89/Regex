## Description

This is a tutorial that explains how a specific regular expression, or regex works by breaking down each part of the expression and describing what it does. The regex used in this tutorial is:

```js

/^[a-z0-9](\.?[a-z0-9]){5,}@g(oogle)?mail\.com$/

```

AS A web development student
